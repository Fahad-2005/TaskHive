# TaskHive
TaskHive is a high-performance task management application built with Flutter and Supabase, featuring a secure, real-time PostgreSQL backend. It allows users to organize work into independent Hives (Workspaces), where they can invite team members and manage access through a robust Row-Level Security (RLS) system. The core experience centers on a dynamic Kanban board where tasks move through custom statuses, and every user action is synced to a unique Profile using automated SQL triggers. By the end of this project, it will be a fully functional, multi-tenant productivity tool capable of handling professional project workflows.
